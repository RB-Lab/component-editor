
# Component-editor

> Web-based editor for components. Initially for React components with
React-Bootstrap preset of components

## Running your project

#### just with live reload:

```bash
$ npm start
```

#### in dev mode (with sourcemaps)

```bash
$ npm run dev
```

#### build:

```bash
$ npm run build
```
